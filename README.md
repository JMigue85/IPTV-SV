# Lista IPTV de Enlaces Públicos

Este repositorio contiene una recopilación de enlaces de transmisión (M3U8) obtenidos de fuentes de acceso público y sitios web oficiales de libre distribución en Internet.

Lista Estándar: IPTV-SV — Máxima compatibilidad.
🛠️ Guía de Optimización y Compatibilidad

Para garantizar la mejor experiencia y estabilidad en las transmisiones, se recomienda seguir estas pautas técnicas:
🚀 Reproductores Recomendados

    FredTV (PC): Reproductor con total compatibilidad. Ofrece el mejor rendimiento y estabilidad.

    MPV Player (PC/Android): Altamente estable. Es la opción ideal como reproductor externo cuando un enlace presenta problemas de carga en otras apps.

    IPTVnator (PC): Se recomienda la siguiente configuración en los ajustes:

        Motor Principal: JSPlayer.

        Motor Opcional: HTML5 (semi-estable).

    Televizo (Android): Compatible con casi toda la lista. Para canales como 2, 4, 6 y SV+, se recomienda usar un reproductor externo (MPV o VLC, en ese orden de prioridad).

⚙️ Ajustes Técnicos y Soluciones

    Decodificación por Software: Se recomienda activar la decodificación por software en MPV y VLC. Esto evita errores de renderizado y mejora la compatibilidad con distintos códecs de video.

    Solución para canales TCS (Token Free): Si los canales de TCS no cargan correctamente en Televizo, utiliza la función "Abrir con reproductor externo" siguiendo este orden:

        MPV Player (Opción principal por estabilidad).

        VLC (Respaldo).

# Public Links IPTV List

This repository contains a curated collection of streaming links (M3U8) gathered from public access sources and official free-to-air websites on the Internet.

Standard List: IPTV-SV — Maximum compatibility.
🛠️ Optimization & Compatibility Guide

To ensure the best experience and stream stability, we recommend following these technical guidelines:
🚀 Recommended Players

    FredTV (PC): Fully compatible player. It offers the best performance and overall stability.

    MPV Player (PC/Android): Highly stable. The ideal choice as an external player if a link fails to load in other applications.

    IPTVnator (PC): The following configuration is recommended in settings:

        Primary Engine: JSPlayer.

        Optional Engine: HTML5 (semi-stable).

    Televizo (Android): Compatible with almost the entire list. For channels such as 2, 4, 6, and SV+, it is recommended to use an external player (MPV or VLC, in that order of priority).

⚙️ Technical Adjustments & Troubleshooting

    Software Decoding: We recommend enabling software decoding in both MPV and VLC. This prevents rendering errors and improves compatibility with various video codecs.

    Solution for TCS Channels (Token Free): If TCS channels do not load correctly in Televizo, use the "Open with external player" function in this order:

        MPV Player (Main option for stability).

        VLC (Backup option).
